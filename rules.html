Grundprinzip:

Die Bodensee MauMau-Regeln halten sich im Grunde an die normalen MauMau-Regeln. Das heißt, man kann gleiche Farbe auf gleiche Farbe und gleiche Zahl auf gleiche Zahl legen. (Bzw. Dame auf Dame und König auf König) Legt ein Spieler seine letzte Karte ab, so sagt er "Mau Mau". Anschließend muss er noch "Matt" sagen, sobald es keine Möglichkeit mehr gibt, dass dieser Spieler doch wieder Karten erhält. Z.B. legt Spieler A seine letzte Karte eine 7 auf den Stapel. Dann sagt er zunächst nur "Mau Mau", da es sein kann, dass alle anderen Spieler auch eine 7 haben und Spieler A daher wieder Karten aufnehmen muss. Es gibt auch noch andere Situationen, bei denen es zu einem ähnlichen Fall kommen kann, insbesondere bei den Karten 5 oder 6. (Mehr dazu später) Das Spiel ist beendet, sobald sich nur noch zwei Spieler mit Karten im Spiel befinden. Wird mit mehr als drei Personen gespielt, so gibt es nicht einen Gewinner sondern mehrere Gewinner, d.h. alle die keine Karten mehr haben. Jedes Spiel hat zwei Verlierer.

Spielvorbereitung:

Es wird mit einem Romme/Canasta-Deck gespielt (2x52 Karten). Ein Spieler beginnt mit dem Austeilen und gibt jedem Spieler 5 Karten. Haben alle Spieler ihre Karten auf der Hand, so deckt der austeilende Spieler die erste Karte vom Stapel auf. Diese wird so gewertet, als hätte sie der austeilende Spieler selbst gelegt. Gespielt wird zunächst im Uhrzeigersinn. Jede Runde muss im Uhrzeigersinn ein neuer Spieler austeilen.

Bedeutung der Karten:

2:
Die 2 kann wie eine normale Karte gespielt werden. Liegt allerdings keine Karte auf dem Stapel, so muss hier zunächst eine 2 gespielt werden. Dann kann ganz normal mit allen Karten weitergespielt werden. Dieser Spezialfall, dass keine Karten auf dem Stapel liegen, wird weiter unten nochmals erklärt.
3:
Die 3 kann immer gespielt werden, auch wenn man selbst nicht am Zug ist. Sie zählt nicht als Zug, d.h. wenn man dran ist und eine 3 legt, dann muss man trotzdem noch seinen Zug spielen. Die 3 ist eine Sonderkarte. Legt man eine 3, so wird die Wirkung der darunterliegenden Karte aufgehoben (eliminiert). Legt einer z.B. eine 7, so kann man die 7 mit einer 3 schnell eliminieren, so dass nicht gezogen werden muss. Für den folgenden Zug ist ist dann die unter der eliminierten Karte liegende Karte maßgeblich. Liegt unter der eliminierten 7 z.B. eine Pik 5, so kann man nun als nächstes Pik oder eine 5 legen. Ein Sonderfall liegt vor, wenn man auf eine 3 noch eine 3 legt. In diesem Fall wird die 3 eliminiert, so dass diese sich aufheben und die unter der ersten 3 liegende Karte wieder Wirkung hat. Eine 3 kann immer gespielt werden, allerdings nicht mehr dann, wenn die Ausführung der unter der drei befindlichen Karte bereits begonnen hat.
4:
Die 4 ist eine Trinkkarte. Sie kann ganz normal gespielt werden, ist aber damit verbunden, dass man einen Schluck trinken muss.
5:
Wird eine 5 gelegt, so müssen alle Spieler die Linke Hand aufeinander auf den Tisch legen. Derjenige, der die 5 gespielt hat, darf seine Hand nicht als erster hinlegen. Wer seine Handals letzter hingelegt hat, muss eine Strafkarte ziehen. Wer eine falsche Hand hingelegt hat, muss zwei Strafkarten ziehen. Demnach zieht der letzte Spieler mit falscher Hand drei Karten vom Stapel.
6:
Die 6 ist die gleiche Karte wie die 5, allerdings mit der rechten Hand.
7:
Die 7 führt dazu, dass der nächste Spieler 2 Karten ziehen muss. Hat dieser Spieler auch eine 7, so kann er die 7 "weitergeben". Der erste Spieler der keine 7 mehr hat, zieht dann für jede 7 auf dem Stapel 2 Karten. Dann ist der nächste Spieler an der Reihe.
8:
Die 8 führt dazu, dass der nächste Spieler aussetzen muss, es sei denn, dieser Spieler verfügt ebenfalls über eine 8. Dann würde wiederum der nächste Spieler aussetzen usw.
9:
Wird eine 9 gelegt, so kann ein beliebiger anderer Spieler eine Karte seiner Wahl, die auf die 9 passt, hinlegen. Auch wenn dieser Spieler nicht an der Reihe wäre. Dieses "Reinwerfen" zählt als Zug, so dass dann der Spieler neben dem, der reingeworfen hat, an der Reihe ist.
10:
Legt man eine 10, so muss man "Miau" sagen. Man kann beliebig viele 10er gleichzeitig legen. Für jede 10 muss dann aber extra "Miau" gesagt werden. Wird das "Miau" vergessen, so muss man eine Strafkarte ziehen. Dabei zählt der Zeitpunkt ab dem der nächste Spieler seinen Zug beginnt.
Bube:
Diese Karte kann auf alle Karten außer die 7 und die 8 gelegt werden, es sei denn diese wurden bereits "entschärft". Man kann sich dann eine Farbe wünschen.
Dame:
Wer diese Karte legt, muss seine Karten mit den Karten eines Mitspielers tauschen.
König:
Wird ein König gelegt, so wird die Spielrichtung gewechselt.
Ass:
Legt ein Spieler ein Ass, so darf er nochmals eine passende Karte darauf legen.
Joker:
Hat man einen Joker auf der Hand, so kann man diesen zu jeder Zeit vor sich legen. (Nicht auf den Ablagestapel) Hat man aktuell die meisten Joker vor sich liegen, so hat man das "Reinwerfprivileg". Das bedeutet, dass man jederzeit eine beliebige aber passende Karte auf den Ablagestapel werfen kann. Hat man eine Karte reingeworfen, so muss man zunächst warten bis man selber am Zug war, um dann wieder eine reinwerfen zu können. Es kann immer nur ein Spieler das Reinwerfprivileg haben. Haben also mehrere Spieler gleich viele Joker vor sich liegen, so erhält derjenige das Privileg, der seinen Joker als letztes abgelegt hat. Hat einer mehr Joker vor sich liegen als andere, so hat dieser automatisch immer das Privileg. Das Reinwerfen zählt wie ein Zug, so dass dann der Spieler neben dem Reinwerfenden an der Reihe ist.(Gemäß Spielrichtung) Muss ein Spieler, der Joker vor sich hat, eine Karte ziehen (sei es, weil er nicht spielen kann oder weil er Strafkarten aufnehmen muss), so muss er pro gezogene Karte einen vor sich liegenden Joker auf den Ablagestapel abgeben. Nun erhält eventuell ein anderer Spieler nach obiger Rangfolge das Privileg.
Ist der Kartenstapel aufgebraucht, so wird der Ablagestapel ohne die oberste Karte zu einem neuen Kartenstapel gemischt.

Strafkarten:

Wer vergisst, "Mau", "Mau Mau", "Miau", oder "Matt" zu sagen, erhält eine Strafkarte. Als Zeitpunkt gilt der Moment, bei dem der nächste Spieler seinen Zug beginnt. Wird auf eine 9 eine nicht passende Karte reingeworfen, so fällt auch eine Strafkarte an. Ebenso, wenn der privilegierte Spieler eine falsche Karte reinwirft.

Besondere Spielsituationen:

Ist die vom Austeiler aufgedeckte Karte eine 3, so ist der Ablagestapel leer. Ebenso wenn die erste Karte von einem anderen Spieler mit einer 3 eliminiert wird. In diesem Fall muss man zunächst eine 2 legen, bevor das Spiel weitergehen kann. Liegen 4 gleiche Karten (von der Zahl her) aufeinander auf dem Ablagestapel, so erhält derjenige, der die 4. Karte gelegt hat einen Pilz. Dieser Spieler kann diesen Pilz einsetzen, um einen anderen Spieler, der bereits ein Gewinner ist (also mit Matt aus dem Spiel geschieden ist), wieder in das Spiel zurückzuholen. Dieser Spieler ist dann nicht mehr Gewinner sondern "stiller Gewinner". Das bedeutet er nimmt ab sofort wieder am Spiel teil, hat jedoch zunächst keine Karten und muss auch nicht ziehen. (d.h. er wird übersprungen). Sobald ein stiller Gewinner jedoch wieder zu einer Karte kommt, sei es durch eine Strafkarte, oder weil ein anderer Spieler seine Karten mit denen des stillen Gewinners getauscht hat, ist dieser wieder als normaler Spieler im Spiel. Man kann also seine Karten mit den (nicht vorhanden) Karten des stillen Gewinners (mit einer Dame) tauschen. Dadurch hat man selbst keine Karten mehr und wird zum Gewinner. Der neue Gewinner muss auch hier "Mau Mau" und "Matt" sagen. Pro Spiel kann es maximal zwei Pilze geben. Ein Spieler (Gewinner) der nicht mehr im Spiel ist, kann keinen Pilz anwenden. Hat sich die Kartenzahl eines Spielers nach dessen Aktion auf eine Karte erhöht/reduziert, so muss dieser "Mau" sagen. Vergessen wird mit einer Strafkarte geahndet. Dieser Fall tritt z.B. auch dann ein, wenn man seine Karten mit einem Spieler tauscht, der nur noch eine Karte hat und man selbst davor mehr Karten hatte. Wird eine 6 auf eine 5 gelegt (oder anders herum), dann müssen alle Spieler beide Hände auf den Tisch aufeinander legen. Der letzte Spieler erhält zwei Strafkarten, wenn seine beiden Hände die letzten sind, ansonsten nur eine Strafkarte.

Präzendenzregeln:

In manchen Situationen kommt es vor, dass man .Mau""Miau""Mau Mau" oder .Matt" gleichzeitig sagen musste. Es ist aber fur diesen Fall eine Rangfolge festgelegt, nach welcher die Worte gesagt werden mussen: Mau " Miau " Mau Mau " Matt Beispiele: Hat man als letzte Karte eine 10 und legt diese ab, so sagt man zuerst .Miau" fur die 10 und dann .Mau Mau" und .Matt". Spezialfall: Hat man nur noch zwei 10er auf der Hand, so kann man diese nach der Regel fur die Karte 10 gleichzeitig legen. Von den beiden 10ern ist allerdings eine 10 die letzte Karte, so dass man sich uberlegen muss, nach welcher Reihenfolge man die Worter spricht. Man uberlegt einfach, man wurde die Karten nacheinander legen: Zuerst legt man also die erste 10. Man musste hier zwei Worte sagen: .Mau" und .Miau". (Da letzte Karte und eine 10). Nach den Prazedenzregeln sagt man zuerst .Mau" und dann .Miau". Nun legt man die zweite 10. Man musste hier drei Worter sagen: .Mau Mau""Miau" und "Matt". Nach den Präzedenzregeln ergibt sich fur diese Karte: "Miau""Mau Mau" und .Matt". Also ware korrekt fur die letzten zwei 10er auf einmal: Mau " Miau " Miau " Mau Mau " Matt Legt man am Ende drei 10er auf einmal, dann ware korrekt: Miau " Mau " Miau " Miau " Mau Mau " Matt

Weiter wichtige Punkte:

Ein Spieler ist unter den Gewinnern, wenn er sein Spiel erfolgreich mit "Matt" beenden konnte. Gewinner können und dürfen nicht mehr am Spielgeschehen teilnehmen. Gewinner können allerdings zu stillen Gewinnern werden. (Siehe Pilz) Stille Gewinner haben keine Karten, nehmen aber am Spiel teil. D.h. sie müssen bei Aktionen wie bei den Karten 5 und 6 mitmachen. Eine 7 führt dazu, dass der stille Gewinner Karten aufnehmen muss und wieder im Spiel dabei ist. Ebenso das Tauschen der Karten mit einer anderen Person. Grob fahrlässige Unruhestiftung oder Verwirrung im Spiel werden mit einer Strafkarte geahndet. Ein Spieler der nicht spielen kann, muss eine Karte ziehen. Danach darf er noch eine beliebige Karte spielen. Ausnahmen siehe Karten 7 und 8.
